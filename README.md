# Portfolio Anime

Portfolio vitrine en une seule page, pensée comme une expérience visuelle cinematique avec effets de scroll, parallaxe legere et reveils d'animation progressifs.

## Apercu

Le projet repose sur trois fichiers principaux:

- `index.html` pour la structure de la page.
- `styles.css` pour l'identite visuelle et les effets d'ambiance.
- `script.js` pour les animations au scroll, la barre de progression et le panneau de debug.

## Fonctionnalites

- Hero fullscreen avec mise en scene immersive.
- Sections portfolio avec reveal au scroll.
- Effets de parallaxe sur les titres, images et formes decoratives.
- Mode `prefers-reduced-motion` pris en charge.
- Panneau de debug activable avec `?debug` ou la touche `D`.

## Lancer le projet

1. Ouvrir le dossier du projet dans VS Code ou dans un navigateur de fichiers.
2. Ouvrir `index.html` dans un navigateur web.
3. Modifier `styles.css` et `script.js` selon vos besoins.
